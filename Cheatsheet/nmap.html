Nmap Cheatsheet

How to install nmap

nmap installation:

Installation
username@machine_name:~$ sudo apt install nmap

Useful tag for nmap


-p $PORT1-$PORT2: Scan from port 1 to port 2
-p $PORT1,$PORT2: Scan port 1 and port 2
-p-: Scann all port
-sV: Scan in order to determine version of the app running on the target
-O: Scan in order to determine the OS running on the target
-sT: TCP connect scan
-sU: UDP scan
-sn: Probe only in order to discorver hosts

Exemple of nmap command

Nmap scan of an network

Network scan
username@machine_name:~$ nmap -O $ip.*.*

Scan all port of an target:

Target scan
username@machine_name:~$ nmap -sV -O -p- $ip

Use a script on the target:

Script
username@machine_name:~$ nmap -p $PORT --script $SCRIPT $IP

Escape the firewall:

firewall escape
username@machine_name:~$ nmap -sV -T1 -f $IP
